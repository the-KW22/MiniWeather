# Mini Weather App ‚õÖ

> A lightweight weather companion with real-time data and air quality monitoring

**Mini Weather App** is a modern, component-based weather application that provides quick access to current weather conditions, detailed forecasts, and air quality information for any location. Built as a learning project to master React, Next.js, and modern frontend development practices.

## ‚ú® Features

- **Current Weather** - Real-time temperature, humidity, wind speed, and conditions
- **5-Day Forecast** - Detailed weather predictions with hourly breakdowns
- **Air Quality Index** - Monitor AQI levels with health recommendations
- **Temperature Charts** - Visual 24-hour temperature trends using Recharts
- **Detailed Weather Stats** - Sunrise, sunset, visibility, pressure, and more
- **Health Recommendations** - Personalized advice based on air quality
- **Responsive Design** - Seamless experience on desktop, tablet, and mobile
- **Smart Caching** - Reduces API calls and improves performance
- **Clean UI** - Modern design with shadcn/ui components
